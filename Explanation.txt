Saara is playing a card arrangement game,
the game is like to rearrange the cards into groups so that each group is size W, and consists of W consecutive cards.
if she can rearrange the print "True" or "False"

Input description:
Card numbers in a row followed by window size as w

Output description:
True or False

Sample Input:
1 2 3 6 2 3 4 7 8
3

Sample Output:
True

Explanation:
[1,2,3],[2,3,4],[6,7,8]
so true

Sample Input:
5 4 6 2 6
3

Sample Output:
False

Explanation:
here we can'y rearrange the numbers as per the window size so the output is "False"
